# Chat App

## Description

Application for exploring Vue.js. Made with Vue.js and Firebase as backend.

## Prerequisites

NodeJS

## Installation

To clone

```
npm install
```

## Compiles and hot-reloads for development

```
npm run serve
```

## To build

```
npm run build
```

Then use the dist folder you get.

## Built With

- vue.js
- CSS
- Materialize.css
- Firebase
